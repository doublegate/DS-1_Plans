**Production approach (Phase 3 decision).** Options ranked by ref-doc fidelity and PDR register:
1. Hand-drawn vector (Illustrator / Affinity) — best publication quality; highest time cost.
2. Mermaid / Graphviz embed + LaTeX figure environments — good for network / fault-tree / sequence diagrams; weaker for cutaways.
3. ASCII diagrams inline in the markdown — lowest quality, not PDR-ready.

Recommended: mixed path. Fault trees and block diagrams in Mermaid (version-controllable, diffable). Cutaways, sphere maps, and schematic cross-sections commissioned as vector art in Phase 3.
